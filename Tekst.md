# Zadatak 2
Napraviti sajt koji prikazuje podatke o svim lansiranjima raketa.

Podaci koje je potrebno prikazati su:
+ Naziv misije
+ Godina misije
+ Datum misije
+ Naziv rakete
+ Mesto lansiranja (launch_site - > site_name)
+ Slika (links -> mission_patch)
